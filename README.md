## Kaggle Competition
https://www.kaggle.com/c/bike-sharing-demand

## RUN

1. Change **"submission_filename"**;

2. Choose model by specifying **"model_name"**.

Model options:
* XGBRegressor;
* GradientBoostingRegressor;
* RandomForestRegressor;
* Lasso;
* LinearRegression.

## REQUIREMENTS
* pandas
* numpy

For _main.py_:
* calendar
* xgboost
* sklearn

For _plots.py_:
* matplotlib
* seaborn
